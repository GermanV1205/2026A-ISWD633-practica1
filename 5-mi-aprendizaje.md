Aprendí varios comandos de docker para diferentes acciones como:
- descargar imagenes .iso
- inspeccionar una imagen 
- eliminar una imagen 
- iniciar y detener un contenedor
- crear y ejecutar un contenedor inmediatamente
- ejecutar comandos dentro de un contenedor de docker en ejecucion
- mostrar el contenido de un archivo con el comando cat
- ver los logs de un contenedor para ver que esta pasando
